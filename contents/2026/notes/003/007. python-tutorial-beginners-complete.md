## Python Basics

### What is Python?

**Python** is:
- **High-Level Language**: Easy to read and write
- **Interpreted**: No compilation needed
- **Dynamically Typed**: Types determined at runtime
- **Multi-Paradigm**: Supports OOP, functional, procedural
- **Versatile**: Web, data science, automation, AI/ML

### Why Python?

**Benefits:**
1. **Easy to Learn**: Simple syntax
2. **Readable**: Code is self-documenting
3. **Versatile**: Many use cases
4. **Large Community**: Extensive support
5. **Rich Libraries**: Vast ecosystem
6. **Career Opportunities**: High demand

### Python Versions

**Python 2 vs Python 3:**
- **Python 2**: Legacy (end of life in 2020)
- **Python 3**: Current version (use this!)
- **Python 3.8+**: Recommended for beginners

### Loops

**For Loop:**
```python

    count += 1
```

**Loop Control:**
```python

    print(i)

for i in range(5):
    print(i)
else:
    print("Loop completed")
```

**Nested Loops:**

**Function with Parameters:**

**Function with Return:**

**Function with Default Parameters:**

```python
    print(f"{greeting}, {name}!")

greet("John", "Hi")        # Hi, John!
```

**Function with Keyword Arguments:**
```python
def person_info(name, age, city):
    print(f"{name} is {age} years old and lives in {city}")

person_info(name="John", age=25, city="New York")
person_info(age=25, city="New York", name="John")
```

**Function with Variable Arguments:**
```python
# *args - variable positional arguments
def sum_numbers(*args):
    total = 0
    for num in args:
        total += num
    return total

result = sum_numbers(1, 2, 3, 4, 5)
print(result)  # 15

# **kwargs - variable keyword arguments
def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="John", age=25, city="NYC")
```

**Lambda Functions:**
```python
# Lambda (anonymous function)
square = lambda x: x ** 2
print(square(5))  # 25

# Lambda with map
numbers = [1, 2, 3, 4, 5]
squared = list(map(lambda x: x ** 2, numbers))
print(squared)  # [1, 4, 9, 16, 25]

# Lambda with filter
even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

**Docstrings:**
```python
def add(a, b):
    """
    Add two numbers.
    
    Args:
        a (int): First number
        b (int): Second number
    
    Returns:
        int: Sum of a and b
    """
    return a + b
```

## Object-Oriented Programming

### Classes and Objects

**Class Definition:**
```python
class Person:
    # Class variable
    species = "Homo sapiens"
    
    # Constructor
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    # Instance method
    def introduce(self):
        return f"My name is {self.name} and I'm {self.age} years old"
    
    # Class method
    @classmethod
    def from_birth_year(cls, name, birth_year):
        age = 2024 - birth_year
        return cls(name, age)
    
    # Static method
    @staticmethod
    def is_adult(age):
        return age >= 18

# Create object
person1 = Person("John", 25)
print(person1.introduce())

# Create using class method
person2 = Person.from_birth_year("Jane", 1995)

# Call static method
print(Person.is_adult(20))  # True
```

### Inheritance

```python
class Animal:
    def __init__(self, name):
        self.name = name
    
    def speak(self):
        return "Some sound"

class Dog(Animal):
    def speak(self):
        return "Woof!"

class Cat(Animal):
    def speak(self):
        return "Meow!"

dog = Dog("Buddy")
print(dog.speak())  # Woof!
```

### Encapsulation

```python
class BankAccount:
    def __init__(self, balance):
        self.__balance = balance  # Private attribute
    
    def deposit(self, amount):
        if amount > 0:
            self.__balance += amount
    
    def withdraw(self, amount):
        if 0 < amount <= self.__balance:
            self.__balance -= amount
    
    def get_balance(self):
        return self.__balance

account = BankAccount(1000)
account.deposit(500)
print(account.get_balance())  # 1500
```

### Polymorphism

```python
class Shape:
    def area(self):
        pass

class Rectangle(Shape):
    def __init__(self, width, height):
        self.width = width
        self.height = height
    
    def area(self):
        return self.width * self.height

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius
    
    def area(self):
        return 3.14 * self.radius ** 2

shapes = [Rectangle(5, 4), Circle(3)]
for shape in shapes:
    print(shape.area())
```

## File Handling

### Reading Files

```python
# Read entire file
with open("file.txt", "r") as file:
    content = file.read()
    print(content)

# Read line by line
with open("file.txt", "r") as file:
    for line in file:
```

### Writing Files

```python

with open("file.txt", "w") as file:
    file.write("Hello, World!\n")
    file.write("This is a new line")

# Append to file
with open("file.txt", "a") as file:
    file.write("\nThis is appended")

# Write multiple lines
lines = ["Line 1\n", "Line 2\n", "Line 3\n"]
with open("file.txt", "w") as file:
    file.writelines(lines)
```

### File Modes

```python
# r - Read (default)
# w - Write (overwrites)
# a - Append
# x - Create (fails if exists)
# b - Binary mode
# t - Text mode (default)
# + - Read and write

# Examples
file = open("file.txt", "r")    # Read
file = open("file.txt", "w")    # Write
file = open("file.txt", "a")    # Append
file = open("file.txt", "rb")   # Binary read
file = open("file.txt", "r+")   # Read and write
```

## Exception Handling

### Try-Except

```python
# Basic exception handling
try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero")

# Multiple exceptions
try:
    value = int("abc")
    result = 10 / value
except ValueError:
    print("Invalid number")
except ZeroDivisionError:
    print("Cannot divide by zero")
except Exception as e:
    print(f"An error occurred: {e}")

# Try-Except-Else
try:
    result = 10 / 2
except ZeroDivisionError:
    print("Cannot divide by zero")
else:
    print(f"Result: {result}")

# Try-Except-Finally
try:
    file = open("file.txt", "r")
    content = file.read()
except FileNotFoundError:
    print("File not found")
finally:
    file.close()  # Always executed
```

### Raising Exceptions

```python
# Raise exception
def divide(a, b):
    if b == 0:
        raise ValueError("Cannot divide by zero")
    return a / b

# Custom exception
class CustomError(Exception):
    pass

def check_age(age):
    if age < 0:
        raise CustomError("Age cannot be negative")
    return age
```

## Best Practices

### Code Style (PEP 8)

```python
# Use meaningful variable names
user_name = "John"  # Good
un = "John"         # Bad

# Use 4 spaces for indentation
def function():
    if True:
        print("Indented with 4 spaces")

# Maximum line length: 79 characters
# Use blank lines to separate functions
def function1():
    pass

def function2():
    pass

# Use docstrings
def add(a, b):
    """Add two numbers."""
    return a + b
```

### List Comprehensions

```python
# Good: List comprehension
squares = [x**2 for x in range(10)]

# Bad: Loop
squares = []
for x in range(10):
    squares.append(x**2)
```

### Context Managers

```python
# Good: Use with statement
with open("file.txt", "r") as file:
    content = file.read()

file = open("file.txt", "r")
content = file.read()
file.close()
```

```python
    elif operator == "*":
        result = num1 * num2
    elif operator == "/":
        if num2 != 0:
            result = num1 / num2
        else:
            return "Cannot divide by zero"
    else:
        return "Invalid operator"
    
    return f"{num1} {operator} {num2} = {result}"

print(calculator())
```

```python
class TodoList:

        self.tasks = []

        self.tasks.append({"task": task, "completed": False})

        if 0 <= index < len(self.tasks):
            self.tasks[index]["completed"] = True
            print(f"Task '{self.tasks[index]['task']}' completed.")
        else:
            print("Invalid task index.")
    
    def show_tasks(self):
        if not self.tasks:
            print("No tasks.")
        else:
            for i, task in enumerate(self.tasks):
                status = "✓" if task["completed"] else " "
                print(f"{i}. [{status}] {task['task']}")
```

## Interview Questions & Answers

### Q1: What is Python?

**Answer:**
- **High-Level Language**: Easy to read and write
- **Interpreted**: No compilation needed
- **Dynamically Typed**: Types determined at runtime
- **Multi-Paradigm**: Supports OOP, functional, procedural
- **Versatile**: Web, data science, automation, AI/ML

### Q2: What is the difference between list and tuple?

**Answer:**
- **List**: Mutable (can be changed), uses []
- **Tuple**: Immutable (cannot be changed), uses ()
- **List**: Slower, more memory
- **Tuple**: Faster, less memory
- **Use List**: When you need to modify
- **Use Tuple**: When you need immutability

### Q3: What is a dictionary?

**Answer:**
- **Key-Value Pairs**: Stores data as key-value pairs
- **Mutable**: Can be changed
- **Unordered**: No guaranteed order (Python 3.7+ maintains insertion order)
- **Fast Lookup**: O(1) average time complexity
- **Unique Keys**: Keys must be unique

### Q4: What is the difference between == and is?

**Answer:**
- **==**: Compares values
- **is**: Compares object identity (same object in memory)
- **Example**: `[1, 2] == [1, 2]` is True, but `[1, 2] is [1, 2]` is False

### Q5: What is a lambda function?

**Answer:**
- **Anonymous Function**: Function without a name
- **Single Expression**: Can only contain one expression
- **Inline**: Often used inline
- **Example**: `lambda x: x**2`

### Q6: What is list comprehension?

**Answer:**
- **Concise Syntax**: Create lists in one line
- **More Readable**: Easier to read than loops
- **Faster**: Generally faster than loops
- **Example**: `[x**2 for x in range(10)]`

### Q7: What is the difference between append() and extend()?

**Answer:**
- **append()**: Adds single element to end
- **extend()**: Adds all elements from iterable
- **Example**: `[1, 2].append([3, 4])` → `[1, 2, [3, 4]]`
- **Example**: `[1, 2].extend([3, 4])` → `[1, 2, 3, 4]`

### Q8: What is __init__?

**Answer:**
- **Constructor**: Special method called when object is created
- **Initialization**: Used to initialize object attributes
- **Self**: First parameter (reference to instance)
- **Example**: `def __init__(self, name): self.name = name`

### Q9: What is the difference between class method and static method?

**Answer:**
- **Class Method**: Takes `cls` as first parameter, can access class
- **Static Method**: No special first parameter, can't access class/instance
- **Class Method**: Decorated with `@classmethod`
- **Static Method**: Decorated with `@staticmethod`

### Q10: What is exception handling?

**Answer:**
- **Try-Except**: Handle errors gracefully
- **Prevent Crashes**: Prevent program from crashing
- **Error Recovery**: Recover from errors
- **Finally**: Code that always executes
- **Example**: `try: ... except Exception as e: ...`
