## 2. Goal Setting and Planning

### 2.1 SMART Goals Framework

**SMART Goals**:
```java
class SMARTGoals {
    
    /**
     * SMART Framework:
     * 
     * S - Specific: Clear and well-defined
     * M - Measurable: Can track progress
     * A - Achievable: Realistic and attainable
     * R - Relevant: Aligned with career goals
     * T - Time-bound: Has a deadline
     */
    
    public Goal createSMARTGoal() {
        // ❌ Bad Goal: "Get better at system design"
        
        // ✅ Good SMART Goal:
        return Goal.builder()
            .specific("Master system design for distributed systems")
            .measurable("Design 5 systems, solve 20 design problems")
            .achievable("With available resources and time")
            .relevant("Required for Staff Engineer role")
            .timeBound("Complete in 6 months")
            .build();
    }
    
    /**
     * Examples of SMART Goals:
     */
    
    public List<Goal> exampleSMARTGoals() {
        return Arrays.asList(
            Goal.builder()
                .specific("Improve code quality")
                .measurable("Reduce bugs by 40%, increase test coverage to 80%")
                .achievable("With current team and tools")
                .relevant("Career growth, team impact")
                .timeBound("Achieve in Q1 2024")
                .build(),
            Goal.builder()
                .specific("Become technical lead")
                .measurable("Lead 2 projects, mentor 3 engineers")
                .achievable("With current experience")
                .relevant("Staff Engineer path")
                .timeBound("Within 12 months")
                .build()
        );
    }
}
```
